Render objects sit below the widget layer and give you direct access to Flutter's layout, painting, and hit-testing pipeline. These guides walk through each major concept in depth — from the base class every render object extends, to the box and sliver subtypes you'll work with most.

## In this section

- [Layout](/guides/layout) — How the layout phase works and what your render object must do during it.
- [RenderObject](/guides/render-object) — The base class all render objects extend and the lifecycle it defines.
- [RenderBox](/guides/render-box) — The 2D box model used by most render objects, including sizing and positioning.
- [RenderSliver](/guides/render-sliver) — Scrollable render objects that efficiently paint only their visible portion.
