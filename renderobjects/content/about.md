**renderobjects.com** is a reference and learning resource for Flutter developers working with custom render objects.

The site covers the full render object API — layout, painting, hit testing, semantics, and debugging — with practical code examples for each method you'll need to override.

## What's here

- **[Guides](/guides)** — Concept-level walkthroughs of how render objects work, from the layout phase to compositing.
- **[API Reference](/api)** — Per-method documentation for every override point in `RenderObject` and `RenderBox`.
- **[Examples](/examples)** — Worked examples of what you can build: custom scrolling, particle effects, data visualization, and game UI.
- **[RenderStudio](/renderstudio)** — A guided tool for generating a custom render object skeleton.
